<h1>NES Emulator</h1>

<h2>Description</h2>
Project consists of C++ software to emulate the function of the Nintendo Entertainment System. Here, the hardware components of the console are re-created with code and used to load and play game ROM (Read-Only Memory) files.
<br />


<h2>Languages and Utilities Used</h2>

- <b>C++</b>
- <b>Dear ImGUI</b>
- <b>SDL</b>
